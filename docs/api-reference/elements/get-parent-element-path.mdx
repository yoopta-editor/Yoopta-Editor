## Overview

The `getParentElementPath` method retrieves the Slate path of an element's parent.

## Signature

```typescript
editor.getParentElementPath(options: GetElementPathOptions): Path | null
```

## Parameters

<ParamField path="options" type="GetElementPathOptions" required>
  <Expandable title="properties">
    <ParamField path="blockId" type="string" required>
      ID of the block containing the element
    </ParamField>

    <ParamField path="element" type="SlateElement" required>
      The element instance
    </ParamField>
  </Expandable>
</ParamField>

## Return Value

Returns the parent `Path` or `null` if not found.

## Examples

```typescript
// Get parent path
const content = editor.getElement({
  blockId: 'accordion-1',
  type: 'accordion-list-item-content',
  path: [0, 1, 1]
});

if (content) {
  const parentPath = editor.getParentElementPath({
    blockId: 'accordion-1',
    element: content
  });
  
  // parentPath will be [0, 1] (the accordion-list-item)
}
```

## Type Definition

```typescript
type GetElementPathOptions = {
  blockId: string;
  element: SlateElement;
};
```
